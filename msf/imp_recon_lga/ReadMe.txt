============ Instruction ============
1. Create a database (for example, "codexworld") and import the "members.sql" file into this database.
2. Open the "dbConfig.php" file and change the $dbHost, $dbUsername, $dbPassword, $dbName variable values with your database credentials.
3. Run the "index.php" file in the browser.
4. Choose the "members.csv" file and click Upload.
5. members.csv file data would be imported and the member's data will be listed.
